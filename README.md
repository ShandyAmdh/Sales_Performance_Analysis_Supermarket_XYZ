# Sales_Performance_Analysis_Supermarket_XYZ

The operations manager asked the data team to create monthly reports to evaluate sales performance and provide business recommendations based on transaction data.

# This report will be presented to:
Store Manager
Finance Manager
Purchasing Manager
Marketing Manager

# Business Objective:
Increase sales and help management make data-driven decisions.

# Business Questions

# Executive Summary:
The manager wants to know:
1. What was the total sales?
2. How many transactions?
3. How many items were sold?
4. What was the average transaction value?

# Sales Performance
1. What is the sales trend over time?
2. Which days have the highest sales?
3. Are there any days with a sharp decline in sales?
4. How are sales growing each week or month (if the data covers that period)?

# Product Performance
1. Which products generate the most sales?
2. Which products are purchased most frequently?
3. Which products contribute the least to sales?
4. Which product categories sell the most?
5. Which categories have the lowest sales?

# Customer Analysis
1. Who are the customers with the largest total purchases?
2. Who are the most frequent customers?
3. What is the average purchase amount per customer?
4. Are there any customers who have the potential to become loyal customers?

# Payment Analysis
1. Which payment methods are most commonly used?
2. What percentage of users use each payment method?
3. Do certain payment methods have higher transaction values?

# Inventory Insight
The Purchasing Manager wants to know:
1. Which products are selling the fastest?
2. Which products are rarely purchased?
3. Which categories need to be stocked more?
4. Which products need to be stocked less?

# Revenue Analysis
The Finance Manager wants to know:
1. Products contributing the most revenue.
2. Categories contributing the most revenue.
3. Top 10 products by revenue.
4. Bottom 10 products by revenue.
